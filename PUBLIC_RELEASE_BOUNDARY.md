# Public Release Boundary

## Public purpose

This repository is a bounded technical-inspection, reproducibility, scholarly-discussion, and citation package. It is not a mirror of the private ECL development repository or connected provisional vaults.

## Included public layer

The advanced release candidate may include:

- public definitions of Root, Template, Phase, Seam, Ledger, closure defect, rupture, phase lock, winding, orientation, and certificate status;
- the closure-before-commit engine and declared thresholds;
- deterministic positive, boundary, and adversarial fixtures;
- native-ledger and PRTP public prototypes;
- schemas, hashes, tests, examples, certificates, limitations, and falsification conditions;
- exact release and source revisions required for reproducibility.

## Excluded private layer

The public repository deliberately excludes:

- unpublished patent specifications, claims, claim charts, examiner strategy, continuation strategy, and filing records;
- private proof-of-existence folders, internal certificate chains, and complete vault history;
- credentials, tokens, account metadata, private endpoints, email addresses, and local configuration;
- local paths, usernames, machine layouts, repository recovery logs, and workstation staging records;
- financial strategies, broker integrations, private datasets, and market systems;
- device fabrication details, laboratory handover packets, private materials work, and unpublished hardware mappings;
- incomplete modules, unresolved imports, debugging artefacts, unrelated experiments, and outputs without clear provenance;
- any claim whose evidence, scientific level, limitations, or public boundary cannot be stated clearly.

## Transport rule

No private file is copied merely for completeness. Public material must be selected, sanitised, tested, provenance-linked, and narrow enough to evaluate the declared public claim without exposing adjacent private inventions.

## Version rule

Every public certificate and result must be tied to an immutable release tag or exact commit. Later changes must not silently replace the meaning of an earlier certificate.

## Rights boundary

Public visibility permits only the uses stated in `LICENSE`. It does not grant a patent licence, waive patent rights, dedicate inventions to the public, or authorise commercial, production, device, derivative, or hosted-service use.
